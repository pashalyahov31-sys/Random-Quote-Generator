## 📖 Описание проекта

**Random Quote Generator** — это десктопное приложение на Python с графическим интерфейсом пользователя (GUI), предназначенное для генерации случайных цитат известных людей. Приложение позволяет не только получать вдохновляющие цитаты, но и сохранять их историю, фильтровать по авторам и темам, а также пополнять базу цитат собственными записями.

### 🎯 Цель создания

- Изучение принципов создания GUI-приложений на Python с использованием Tkinter
- Практика работы с файловой системой (JSON) для сохранения данных
- Освоение системы контроля версий Git
- Разработка полезного инструмента для повседневного вдохновения

### 👥 Целевая аудитория

- Пользователи, ищущие ежедневную порцию мотивации
- Блогеры и писатели, нуждающиеся в источниках вдохновения
- Студенты, изучающие программирование на Python
- Преподаватели для демонстрации принципов ООП и работы с файлами

## ⚡ Функциональные возможности

### Основной функционал

| Функция | Описание | Статус |
|---------|----------|--------|
| 🎲 Генерация случайной цитаты | Выбор случайной цитаты из базы данных | ✅ |
| 📝 Добавление новых цитат | Возможность пополнять базу собственными цитатами | ✅ |
| 📜 История просмотра | Сохранение всех сгенерированных цитат с меткой времени | ✅ |
| 🔍 Фильтрация по автору | Отображение только цитат выбранного автора | ✅ |
| 🏷️ Фильтрация по теме | Отображение только цитат выбранной тематики | ✅ |
| 💾 Автосохранение | Данные сохраняются между сеансами работы | ✅ |
| 🧹 Очистка истории | Возможность очистить всю историю цитат | ✅ |
| ✅ Валидация ввода | Проверка на пустые поля при добавлении цитат | ✅ |

### Дополнительные возможности

- **Счётчик цитат** — отображение общего количества цитат в базе
- **Статистика истории** — сколько записей отображается после фильтрации
- **Красивое форматирование** — удобное отображение цитат в интерфейсе
- **Прокрутка истории** — поддержка прокрутки для длинных списков
