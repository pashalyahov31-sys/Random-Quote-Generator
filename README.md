## Описание

Random Quote Generator — это десктопное приложение на Python с графическим интерфейсом (Tkinter), которое позволяет:

- Генерировать случайные цитаты из предустановленного списка.
- Добавлять новые цитаты (текст, автор, тема) с проверкой на пустые поля.
- Просматривать историю всех сгенерированных цитат.
- Фильтровать историю по автору и теме.
- Сохранять цитаты и историю в JSON-файлы для сохранения данных между сеансами.
